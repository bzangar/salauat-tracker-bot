Функции:

- войти в аккаунт
- посмотреть мои салауаты на сегодня
- посмотреть мои салауаты на неделю
- посмотреть салауаты других за неделю
- топ 3 за текущий месяц по салауатам

---

Таблицы:

- User:

  - id
  - username
  - tgName
  - password
  - salauats

- Salauat

  - id
  - user_id
  - count
  - date

  ***

Логика:

- Логика записи салауатов:
- /login - пользователь заходит с свой аккаунт через логин и пороль
- /logout - пока не вызовет команду все операции будут делаться под его именем
- /add - при этой команде пользователя просят ввести количество салауатов. Если за сегодня не было то создается новая запись в таблице Salauat, а если уже есть за сегодня запись то она увеличивается
- /edit - при этой команде пользователь меняет кол. салауатов за сегодня
- /today - салауаты всех за сегодня. По убыванию кол. салауатов
- /week - салауаты всех за неделю. По убыванию кол. салауатов
- /month - салауаты всех за месяц. По убыванию кол. салауатов
- в 23:59 все записи в базе пишутся в excel файл
- (в дальнейшем) рейтинг по ударному режиму
